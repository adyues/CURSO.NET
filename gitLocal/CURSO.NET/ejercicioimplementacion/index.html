Implementacion De Soluciones De Movilidad con Tecnologias Web

Objetivos

El desarrollo web certificado para la W3C para moviles se ha convertido en una alternativa al desarrollo de aplicaciones nativas para los diferentes dispositivos por si simplicidad y ahorro de costes, facilitando la creacion de aplicaciones para diferentes dispositivos moviles.

Prerequisitos

Estar familiarizado con el desarrollo de aplicaciones y/o paginas web y las herramientas TIC
